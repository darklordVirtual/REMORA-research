Policy Outcomes
ACCEPT
Evidence strong, consensus sufficient, policy permits. Return answer with full Merkle audit trace.
VERIFY
Uncertainty manageable. Retrieve more evidence or request additional oracle sweep before acting.
ABSTAIN
Trust < 0.12. Three independent oracles disagree severely. Deferring is better than forcing an answer.
ESCALATE
Risk or authority exceeds automation boundary. Human reviewer receives full evidence + EscalationPayload.

Honest limits — what is not yet demonstrated
N=544 temperature threshold is in-sample calibrated — not an independent holdout result. Tool-call benchmarks are deterministic heuristic simulators, not live LLM calls. EvidenceOracleV3 uses lexical (Jaccard) verification, not semantic entailment. Conformal 5% risk target fails all 20 repeated splits on N=302. Governance drift thresholds require operational telemetry before enforcement. No production environment has been validated.
